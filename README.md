# Hiroshima.rb

この gem は Hiroshima.rb が提供する gem です。
基本的にネタです。
まずは hiroshimarb コマンドを提供したいと思いますが、未定です。

## Installation

    $ gem install hiroshimarb

## Usage

### user コマンド

    $ hiroshimarb open

ブラウザで Hiroshima.rb のウェブサイトを表示します。

### member コマンド

    $ hiroshimarb member

Hiroshima.rb のメンバー一覧を表示します。
