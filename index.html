Hiroshima.rb

この gem は Hiroshima.rb が提供する gem です。 基本的にネタです。 まずは hiroshimarb コマンドを提供したいと思いますが、未定です。

Installation

$ gem install hiroshimarb

Usage

open コマンド

$ hiroshimarb open

ブラウザで Hiroshima.rb のウェブサイトを表示します。

open コマンドにはサブコマンドが用意されていて、指定するサブコマンドによって開くサイトが変化します。 用意されているサブコマンドは以下の通りです。

hiroshimarb Hiroshima.rb
oc-h おしい広島県

member コマンド

$ hiroshimarb member

Hiroshima.rb のメンバー一覧を表示します。
